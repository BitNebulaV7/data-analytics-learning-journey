## 🚀 About This Project

A structured, project-based learning journey for becoming a **Data Analyst**.

This repository tracks my hands-on practice with:
- 📊 Excel (data cleaning, formulas, pivot tables)
- 🗄️ SQL (queries, joins, aggregations)
- 🐍 Python (Pandas, NumPy, visualization)
- 📈 Power BI (dashboards and reporting)

All skills are applied on real-world datasets (starting with **Sample Superstore**).

## 🎯 Goal

Build real-world data analysis skills through:
- Daily practice
