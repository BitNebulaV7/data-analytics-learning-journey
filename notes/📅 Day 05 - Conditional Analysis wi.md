# 📅 Day 05 - Conditional Analysis with Excel

## 🎯 Topic

Using logical functions in Excel for data classification and analysis.

---

## 📚 What I Learned

Today I practiced using conditional logic in Excel to create new insights from existing data.

Topics covered:

- IF function
- AND function
- OR function
- Conditional classification
- Creating business rules with logical conditions

---

## 🛠 Practice

Worked with the Sample Superstore dataset.

Created new analytical columns:

- Profit Category
- Discount Risk
- Order Risk

Used conditions to classify records based on:

- Profit values
- Discount levels
- Multiple conditions combined together

---

## 💡 Key Takeaways

- IF statements are not only for automation; they can help transform raw data into meaningful categories.
- Business rules can be translated into logical conditions.
- Creating categories makes it easier to identify patterns and risky areas in a dataset.

---

## 📊 Business Insights

- High sales do not always mean high profitability.
- Some orders may require investigation when high discounts are combined with negative profit.
- Conditional analysis helps identify areas that need deeper analysis.

---

## 🤔 Reflection

Today I learned that the important part of conditional analysis is not writing the IF formula itself, but defining meaningful rules and asking the right business questions.

The same logic used in programming can be applied to analyze and categorize data in Excel.
