# 📅 Day 03 - Sorting & Filtering in Excel

## 🎯 Topic

Sorting and Filtering Data

---

## 📚 What I Learned

Today I learned how to organize and explore data using Excel's built-in sorting features.

Topics covered:

- ✅ Accessing Sorting Options
- ✅ Sorting Text, Numbers, and Dates
- ✅ Sorting by Cell Color / Font Color
- ✅ Multi-Level Sorting
- ✅ Custom List Sorting
- ✅ Left-to-Right Sorting
- ✅ Case-Sensitive Sorting
- ✅ Restoring the Original Sort Order
- ✅ Common Sorting Issues
- ✅ Partial Sorting (Last Name)

---

## 💡 Key Takeaways

- Sorting is not just about ordering data—it helps reveal patterns and outliers.
- Multi-level sorting allows more meaningful organization of datasets.
- Excel can sort using colors and custom lists, not only values.
- Keeping the original order before sorting is important when exploring data.
